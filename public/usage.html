Holo Vault Usage

Fast path: Search → cards auto-tick → green Import selected → wait. No need to click each Add button.

Add many cards

  1. Search a set name or card (e.g. surging sparks).
  2. New cards are checked automatically.
  3. Set Default × (1.0 = market price in NZD).
  4. Click Import selected (N) and keep the tab open (~3 sec/card).

Multiple searches (queue)

  1. Search set A → tick cards → Add selected to queue.
  2. Search set B → tick more → Add selected to queue again.
  3. Bottom bar → Import queue to Shopify.

Troubleshooting

  • Rate limit — wait 30s; don’t sync during import.
  • All new (0) — everything already listed.
  • Auth error — check Shopify client ID/secret in Railway or .env.
